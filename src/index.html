¿Qué es esta web?

Esta web reúne varios apartados relacionados con mi vida académica, mis intereses personales y el trabajo realizado en la asignatura de Fundamentos de Ingeniería Informática. Su objetivo es servir como una pequeña carta de presentación y como práctica de HTML y CSS.

Esta web reúne una serie de apartados propuestos por el enunciado del proyecto, en los cuales encontramos el primer apartado de la página "Sobre mi", seguimos con el grado que estoy estudiando "Ingeniería Informática", una página reservada para el Sporting de Gijón, y dos últimos apartados de "net" y "contacto".
La página esta creada como práctica de HTML y CSS. Con diferentes etiquetas y estilos de clase.
